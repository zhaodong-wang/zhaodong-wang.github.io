Service-level Traffic Analysis

Facebook Network Infrastructure

Designed an end-to-end framework for modeling and simulating service-level traffic across Facebook’s production network. Built a failure simulator that projected how topology changes would affect capacity, and contributed to real-time monitoring and service draining workflows used for safe migrations.

Failure Prediction by Machine Learning

Built SVM-based classifiers to predict infrastructure failures from sensor telemetry, with a secondary optimization pass that recommended minimum-cost sensor placements for target detection coverage.

Locomotive Footprint Project

CSX Transportation

Multi-year collaboration with CSX on jointly optimizing the location, routing, and inventory of locomotive servicing facilities. Formulated as a large-scale integer program with custom decomposition. Resulted in a published paper in Interfaces and an INFORMS Wagner Prize finalist nomination.

Bullwhip Effect Analysis

Analytical work on the propagation of demand variability in supply chains under a class of nonlinear ordering policies, proving bounded-growth conditions and publishing the results in the European Journal of Operational Research.

Emission Mitigation of Intelligent Vehicles

Applied model predictive control to coordinate the longitudinal dynamics of autonomous vehicles inside a congested platoon, reducing fuel consumption and emissions across the convoy. Published in Computer-Aided Civil and Infrastructure Engineering.
